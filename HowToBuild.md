# How to build
### Requirements
- OS: Windows 10 or higher or Ubuntu 20.04 or higher
- Memory: 2 GB or more
- Disk space: At least 100MB of free disk space
- [Node.js](https://nodejs.org/en/) (Version 16.x)
### 1. install dependencies
```bash
npm install
```
### 2. build
```bash
npm run build
```
It is generated in the dist folder.
